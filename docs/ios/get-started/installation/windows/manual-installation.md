Jeśli nie masz jeszcze Xamarin lub Visual Studio, wykonaj kroki [tutaj](https://msdn.microsoft.com/en-us/library/mt613162.aspx) Pobierz i zainstaluj program Xamarin dla Visual Studio 2015.
