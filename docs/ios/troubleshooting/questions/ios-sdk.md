# <a name="where-can-i-set-my-ios-sdk-locations"></a>Gdzie mogę ustawić Moje iOS lokalizacje zestawów SDK?

## <a name="visual-studio"></a>Visual Studio

**Narzędzia -> Opcje -> Xamarin -> Ustawienia systemu iOS**

## <a name="visual-studio-for-mac"></a>Visual Studio for Mac

**Program Visual Studio for Mac -> Preferencje -> Projekty -> Lokalizacje zestawów SDK -> dla systemu iOS**

## <a name="default-ios-sdk-paths"></a>Domyślne ścieżki zestawu SDK dla systemu iOS

Xamarin.iOS i zestawu Apple SDK (Xcode) mają domyślną lokalizację, gdy pola Lokalizacja jest pusty. Te ustawienia domyślne są następujące:

- Środowisko Xcode: `/Applications/Xcode.app`
- Xamarin.iOS (Visual Studio dla komputerów Mac): `/Library/Frameworks/Xamarin.iOS.framework/Versions/Current`
